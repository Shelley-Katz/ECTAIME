# Benchmark Selection Rubric v1

Use this rubric when adding or replacing benchmark clips.

## Required Coverage

1. Sustained lyrical phrase.
2. Fast connected scalar phrase.
3. Repeated-note ostinato phrase.
4. Mixed articulation boundary phrase.
5. Rest-heavy phrase.
6. Dense divisi context phrase.

## Clip Quality Rules

1. Clip should contain clear musical intent.
2. Clip should not rely on known score defects unless tagged explicitly.
3. Clip must have stable source alignment metadata.

## Acceptance Rules

1. New clip IDs must be unique.
2. Category and expected primary profile must be specified.
3. Any replacement clip must preserve category coverage.
