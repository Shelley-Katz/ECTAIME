# ECT Benchmarks

This folder stores the frozen benchmark corpus for objective tuning and gate decisions.

Core files:

1. `benchmark-manifest.v1.yaml`
2. `selection-rubric.md`

Subfolders:

1. `clips/` for optional extracted excerpt files.
2. `notes/` for listening annotations.

Rule:

1. Benchmark definitions are frozen per version.
2. Tuning claims must reference benchmark IDs.
